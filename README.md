# Docker development environment

PHP 8, PostgreSQL and NGINX docker development environment

## PHP version

php-fpm 8.2

## PostgeSQL version

PostgreSQL 15

## Additional packages

- Composer

- NPM

- cURL

- Xdebug

# Run

$ make build

$ make up

# Stop

$ make Stop

# Restart

$ make Restart

# For execute console command use

$ ./exec
